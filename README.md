# 📊 User Profile & Daily Habit Analysis Dashboard

## 📌 Overview
This project presents an interactive Power BI dashboard built using two datasets: **User Profiles for Ads** and **Daily Habit Tracker**. The dashboard provides insights into user demographics, online behavior, education, income, and daily lifestyle habits through dynamic visualizations.

## 🚀 Features
- Total Users KPI
- Total Locations
- Language Distribution
- Average Education Level
- Average Income Level
- Average Age
- Average Study Hours
- Average Sleep Hours
- Average Steps
- Average Calories Burned
- Interactive Filters (Date, Gender, Location, Language, Education, Income)
- Dynamic Charts and KPI Cards

## 📂 Dataset
- User Profiles for Ads
- Daily Habit Tracker

## 📈 Visualizations Used
- KPI Cards
- Donut Chart
- Pie Chart
- Gauge Chart
- Bar Chart
- Stacked Column Chart
- Slicers

## 🛠 Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## 🎯 Dashboard Insights
- Analyze user demographics and interests.
- Compare education, income, and language distributions.
- Monitor study hours, sleep hours, and physical activity.
- Explore user behavior through interactive filtering.
